# Science Festivity 2025 - Arab Scientists of the Golden Age of Islam

A comprehensive educational website showcasing the contributions of Arab scientists during the Golden Age of Islam.

## Features

- **Interactive Map**: Explore different tents and areas of the science festivity
- **Educational Games**: 
  - Quiz game with multiple levels
  - Matching game for scientists and their discoveries
  - Ping Pong game
- **Chatbot Assistant**: "Eisho" - an AI assistant with knowledge about Arab scientists
- **3D Background**: Beautiful arabesque patterns using Three.js
- **Multilingual Support**: Arabic and English interface

## Structure

- `index.html` - Main homepage
- `pages/` - Different sections (tents, games, map, etc.)
- `GAME 1/` - Ping Pong game
- `game2/` - Scientists matching game
- `js/` - JavaScript files for functionality
- `css/` - Styling files
- `assets/` - Images, videos, and other media files

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Three.js for 3D graphics
- Font Awesome for icons
- Google Fonts (Cairo)

## Deployment

This site is configured for GitHub Pages deployment. The site will automatically deploy when changes are pushed to the master branch.

## Local Development

Simply open `index.html` in a web browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .

# Using PHP
php -S localhost:8000
```
